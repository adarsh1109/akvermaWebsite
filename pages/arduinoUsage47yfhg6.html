Arduino Usage

SENDING DIGITAL TRIGGERS TO EEG VIA SERIAL PORT in MATLAB

                   
               In Arduino IDE, put the following command:

               void setup() {
                 Serial.begin(19200);
                 Serial.flush();
                 pinMode(LED_BUILTIN, OUTPUT);
               }
               
               void loop() {
                 if (Serial.available() > 0){
                   int val = char(Serial.read())-'0';
                   Serial.println(val);
                   Serial.write("\n");
                   if(val == 1){
                     Serial.write("ON\n");
                     digitalWrite(13,HIGH);
                     delay(0001);
                     digitalWrite(13,LOW); 
                     }
                   
                 }
               }
               
               In MATLAB
               ser = serialport("/dev/cu.usbserial-210",19200)
               writeline(ser,’1’)

SENDING DIGITAL TRIGGERS TO EEG VIA SERIAL PORT in OPENSESAME

          
               Install pip and pyserial using terminal (for Mac)
               1. Get python version: % python3 --version
               2. Install pip: python3 get-pip.py
               3. Add pip to the path: echo 'PATH="/Users/ak.verma/Library/Python/3.9/bin"'
               4. Verify pip installation using: pip3 --version
               5. Upgrade pip if required
               6. Install pyserial if not already installed: pip3 install pyserial
               7. Check for pyserial in pip list: pip3 list
               8. Add python to zsh if required: echo "alias python=/usr/bin/python3" >> ~/.zshrc
               9. Check for serial ports: python -m serial.tools.list_ports
               
               
               In OpenSesame (python) 
               import serial
               dev = serial.Serial("COM4", baudrate=19200)
               dev.write(b'1')

MAC- Psychtoolbox HID error (PsychHID-ERROR)

                   1. Quit Matlab 64-Bit, and also Terminal if it was used to launch Matlab 64-Bit.
               2. Open 'System Preferences', then go to the 'Security & Privacy' control panel.
               3. Select the 'Privacy' tab, and then in the list on the left, the item 'Input Monitoring'.
               4. Remove MATLAB from the list on the right by clicking '-' button.
               Add MATLAB again by clicking '+' button.
               Check the box next to MATLAB.
               5. Make sure that the checkbox next to the item's name is checked.
               6. Restart Matlab 64-Bit.
               7. Execute the 'KbCheck' command as a simple test, hopefully working now without errors.
